Aim 2: Predicting Amyloid PET Positivity from Plasma Biomarkers

Overview

This analysis predicts amyloid PET positivity from plasma biomarkers (ptau181, ptau217) using logistic regression. We calculate:

  • ROC curves and AUC
  • Optimal cutoffs using Youden index (sensitivity + specificity - 1)
  • Sensitivity and specificity at optimal cutoffs

Amyloid positivity thresholds:

  • Centiloid ≥ 22.32 (PREVENT-AD cohort-specific)
  • Centiloid ≥ 18 (Sensitivity analysis)
